
# Retail Demand Forecasting & Inventory Replenishment Optimization

This project analyzes retail sales and inventory data to identify stockout patterns, forecast demand, and recommend inventory replenishment quantities.

## Project Components

- ETL pipeline to transform raw retail datasets
- Exploratory data analysis of sales and inventory trends
- Demand forecasting using historical sales data
- Inventory replenishment model using safety stock and reorder point calculations

## Technology Stack

Python  
Pandas  
NumPy  
Matplotlib  
Jupyter Notebook
